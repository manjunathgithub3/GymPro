# Membership Use Cases

Version: 1.0
Status: Draft
Project: GymPro

## Overview
This document defines membership-related use cases for GymPro.

## Primary Actors
- Staff/Admin
- Member
- System

## Use Cases

### UC-MSHIP-001: Create Membership Plan
- Description: Admin defines membership packages and pricing.

### UC-MSHIP-002: Buy Membership
- Description: Member purchases a membership plan.

### UC-MSHIP-003: Renew Membership
- Description: Member renews an existing membership.

### UC-MSHIP-004: Freeze Membership
- Description: Member pauses membership temporarily.

### UC-MSHIP-005: Cancel Membership
- Description: Member cancels their membership.
