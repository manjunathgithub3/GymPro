# Reports Use Cases

Version: 1.0
Status: Draft
Project: GymPro

## Overview
This document defines reporting use cases for GymPro.

## Primary Actors
- Admin
- Branch Manager
- Finance Team

## Use Cases

### UC-REP-001: Generate Membership Report
- Description: Admin generates a report of active and expired memberships.

### UC-REP-002: Generate Revenue Report
- Description: Finance team views revenue details by date or branch.

### UC-REP-003: Generate Attendance Report
- Description: User reviews attendance summaries for a selected period.

### UC-REP-004: Export Report
- Description: User exports a report in Excel or PDF format.
