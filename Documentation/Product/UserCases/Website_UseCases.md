# Website Use Cases

Version: 1.0
Status: Draft
Project: GymPro

## Overview
This document defines website CMS and public-facing use cases for GymPro.

## Primary Actors
- Admin
- Visitor
- Member

## Use Cases

### UC-WEB-001: View Public Website
- Description: Visitor opens the gym website and views public pages.

### UC-WEB-002: Submit Contact Form
- Description: Visitor sends a message or inquiry through the website.

### UC-WEB-003: Manage Website Content
- Description: Admin updates pages, services, and banners.

### UC-WEB-004: Display Membership Offers
- Description: Website shows membership plans and promotional campaigns.
