# 1. Introduction

This document defines all functional modules of GymPro.

Each module contains one or more business features.

Every feature is assigned a priority.

Priority Levels

P0 = Critical (Application cannot function)

P1 = MVP Feature

P2 = Future Enhancement

P3 = Long-Term Feature
